## 飞书联系人

通过 open_id 识别飞书用户（群消息前缀 `[sender=ou_xxx]`）。

**识别规则：**
- 遇到不认识的 open_id，先在表中记录为「未知」，**不要强行匹配**到已知联系人
- 如果对方自我介绍或从上下文能确定身份，再更新名字和备注

| open_id | 称呼 | 备注 |
|---------|------|------|
| (从收到的消息中获取) | | |

## 群聊

| chat_id | 群名 | 类型 | 氛围 | 成员 |
|---------|------|------|------|------|
| | | 工作群/朋友群 | | |

## @ 提及格式

在飞书中 @ 人：`<at user_id="ou_xxx">用户名</at>`

## 维护规则

1. **进入新群聊后**：立即记录 chatid 到群聊表
2. **了解群成员后**：更新成员信息
3. **知道 open_id 对应的名字后**：立即记录到联系人表
